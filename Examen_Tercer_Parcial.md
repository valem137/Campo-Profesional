# Comandos básicos de terminal

| Comando   | Funcion |
|-----------|-------------------|
| `dir`     | Lista archivos y carpetas del directorio actual. |
| `cd`      | Cambia de carpeta o muestra la ruta actual. |
| `cls`     | Limpia la pantalla de la terminal. |
| `copy`    | Copia archivos a otra ubicación. |
| `move`    | Mueve archivos o carpetas. |
| `del`     | Elimina archivos. |
| `mkdir`   | Crea una nueva carpeta. |
| `rmdir`   | Elimina una carpeta vacía. |
| `ipconfig`| Muestra configuración de red (IP, gateway, etc.). |
| `ping`    | Comprueba conexión enviando paquetes a una dirección. |


# Comandos básicos de Git/GitHub 

| Comando              | Descripción breve |
|----------------------|-------------------|
| `git init`           | Inicializa un nuevo repositorio en la carpeta actual. |
| `git clone <url>`    | Clona un repositorio remoto a tu equipo. |
| `git status`         | Muestra el estado de los archivos (modificados, no rastreados, etc.). |
| `git add <archivo>`  | Agrega archivos al área de preparación (staging). |
| `git commit -m ""`   | Guarda los cambios con un mensaje. |
| `git push`           | Envía los cambios al repositorio remoto. |
| `git pull`           | Descarga y fusiona los últimos cambios del repositorio remoto. |
| `git branch`         | Lista ramas o crea nuevas dependiendo de los parámetros. |
| `git checkout <rama>`| Cambia de rama. |
| `git merge <rama>`   | Fusiona otra rama con la rama actual. |
