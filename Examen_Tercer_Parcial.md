# Comandos básicos de terminal

| Comando  | Función |
|-----------|-------------------|
|__`dir`__   | Lista archivos y carpetas del directorio actual. |
| __`cd`__ ; __`cd ..`__     | __`cd`__ cambia de carpeta o muestra la ruta actual. __`cd ..`__  sirve para subir un nivel en la jerarquía de directorios. |
| __`cls`__     | Limpia la pantalla de la terminal. |
| __`copy`__    | Copia archivos a otra ubicación. |
| __`move`__    | Mueve archivos o carpetas. |
| __`del`__     | Elimina archivos. |
| __`mkdir`__   | Crea una nueva carpeta. |
| __`rmdir`__   | Elimina una carpeta vacía. |
| __`ipconfig`__| Muestra todos los valores actuales de configuración de red TCP/IP y actualiza la configuración del Protocolo de configuración dinámica de host (DHCP) y del sistema de nombres de dominio (DNS). |
| __`ping`__    | Comprueba conexión enviando paquetes a una dirección. |
|__`rename`__| Cambia el nombre de un archivo.|
|__`systeminfo`__|Muestra información de configuración detallada sobre un equipo y su sistema operativo, incluida la configuración del sistema operativo, la información de seguridad, el identificador de producto y las propiedades de hardware.|
|__`find`__|Busca una cadena de texto en un archivo o archivos y muestra líneas de texto que contienen la cadena especificada. |

[Comandos de Windows](https://learn.microsoft.com/es-es/windows-server/administration/windows-commands/windows-commands)

# Comandos básicos de Git/GitHub 

| Comando              | Función|
|----------------------|-------------------|
| __`pwd`__ | Devuelve la ruta de la carpeta actual. |
| __`cd`__ ; __`cd ..`__ | Al igual que en la terminal de Windows, sirve para cambiar entre directorios.  |
|__`ls`__| Muestra el contenido de los directorios en la ubicacion actual.|
|__`ls -la`__| Muestra el contenido con informaciion adicional (permisos).|
|__`mkdir`__| Crea una nueva carpeta. |
|__`touch`__| Crea un archivo vacio si no existe, si ya existe actualiza sus marcas de tiempo de acceso y modificación a la hora actual del sistema, sin modificar el contenido del archivo.|
|__`vim`__|Abre y edita un archivo desde terminal.|
|__`:wq`__|Escribe y sale del archivo luego de editarlo.|
| __`git init`__| Inicializa un nuevo repositorio en la carpeta actual. |
| __`git clone <url>`__| Clona un repositorio remoto a tu equipo. |
| __`git status`__| Muestra el estado de los archivos (modificados, no rastreados, etc.). |
| __`git add <archivo>`__| Agrega archivos al área de preparación (staging). |
| __`git commit -m ""`__| Guarda los cambios con un mensaje. |
| __`git push`__| Envía los cambios al repositorio remoto. |
| __`git pull`__| Descarga y fusiona los últimos cambios del repositorio remoto. |
| __`git branch`__| Lista ramas o crea nuevas dependiendo de los parámetros. |
| __`git checkout <rama>`__| Cambia de rama. |
| __`git merge <rama>`__| Fusiona otra rama con la rama actual. |

[Comandos de Git](https://git-scm.com/cheat-sheet)
